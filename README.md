# Annotation Console

## Tech Stack

- Next.js 15
- React
- TypeScript
- Redux Toolkit
- Tailwind CSS
- Axios
- IndexedDB
- WebSocket
- Server-Sent Events

## Setup

### Install

```bash
npm install
```

### Run mock server

```bash
cd mock-server
npm install
npm run mock
```

### Run application

```bash
npm run dev
```

## Features

- Task List
- Search
- Status Filter
- Type Filter
- Sorting
- Pagination
- WebSocket Live Updates
- Streaming Markdown Summary
- IndexedDB Cache
